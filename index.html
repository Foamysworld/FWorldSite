What is FMission?

FMission is a lightweight, easy to use, super flexible modular Mission system written for DayZ Epoch that allows for robust Mission creation through easy to use templates and tools. FMission runs from the built in Epoch Scheduler so it's easy to install and configure. FMission is written to be an independant system that won't interfere with other systems and is less likely to break when Epoch updates.

FMission Features

  • Run up to 3 missions at once
  • Robust AI creation tools
  • AI Unit, Vehicle and Heli Patrols
  • Refreshing Markers with Debug capability
  • Unlimited Random Mission Selection
  • Random Minefield capability
  • Random Mission Spawn Chance
  • Custom Loot Boxes per Mission
  • Runs independant of all AI/Mission systems
  • Adjustable Timeouts
